# link-pilot 1분 데모 플로우

## 목표
북마크 저장부터 요약/정리까지 1분 안에 확인.

## 시나리오
1. 서버 실행
2. 웹 페이지 2개 열기
3. 각 페이지에서 `⌘ + ⌥ + ⇧ + B`로 저장
4. 팝업 열고 unread 목록 확인
5. 첫 항목 `요약/재시도` 클릭
6. 두 번째 항목 태그 `ai,tool` 입력 후 저장
7. 태그 칩 `#ai` 클릭해서 필터 확인
8. `Export`로 백업 파일 저장

## 데모 성공 기준
- 저장 2건 생성
- 요약 문장/포인트 표시
- 태그 필터 정상
- Export 파일 다운로드 완료
