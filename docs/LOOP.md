# Ralph Loop (설계↔구현 짧은 반복)

## Loop Cadence
- L0 (30~60분): 문제/가정 정의
- L1 (60~90분): 가장 작은 동작 단위 구현
- L2 (20분): 실제 사용 시나리오 검증
- L3 (15분): 수정/정리 후 다음 루프로 이동

## 이번 프로젝트 적용
1. Loop A: 저장 API + DB insert
2. Loop B: 익스텐션 단축키 저장 호출
3. Loop C: unread 리스트 UI
4. Loop D: 요약 요청 버튼/큐 인터페이스

## DoD (각 루프)
- 데모 가능한 상태
- 로그/에러 확인 경로 명시
- 다음 루프 진입 조건 1줄 정리
